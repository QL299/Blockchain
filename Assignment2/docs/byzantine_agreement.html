byzantine_agreement.simple_ba module¶

class byzantine_agreement.simple_ba.SimplePKIBA(sender)¶

Bases: object

A simple PKI based protocol for Byzantine agreement.

Parameters:sender (int) – Node ID of sender in peer list.
s_i¶

list of tuples of str – represents accepted proposals.

votes¶

map of str to set of int – represents nodes that broadcasted signatures on a message/proposal.

signatures¶

map of str to list of (tuple of int, str) – maps string messages/proposals to accepted signatures; each signature is a tuple of the node ID that signed and string hex signature.

sender¶

int – node ID acting as the protocol sender (see protocol description in notes)

curr_round_number¶

int – last round processed internally by the BA protocol

broadcast_votes_for(round, votes)¶

Broadcast votes on a proposal to all nodes; this happens once a proposal is added to s_i.

calculate_votes_for(round)¶

Calculate what votes a node should gossip out in the provided round. These are proposals that match the criteria given in 4.1, and that a node has not previously added to its s_i set and broadcast votes for. This function should also update the relevant data structures with the current node’s vote.

Parameters:round (int) – Round to target.
Returns:Returns a list of proposals to broadcast votes for.
Return type:list of str
get_output()¶

Returns the final output of agreement once the protocol has completed, and None before then.

get_proposals_with_threshold(round)¶

Gets proposals that have reached the threshold required by a given round.

Parameters:round (int) – Round to target.
Returns:Returns a list of all proposals eligible for adding to a node’s s_i (proposals that have achieved required vote thresholds).
Return type:list of str

This function DOES NOT need to check signatures; assume they are already checked in process_vote.

is_done()¶

Returns True once the protocol has completed, and False before.

process_vote(vote)¶

Process an incoming vote and add to relevant datastructures.

run_protocol_loop()¶

Runs the protocol loop; tracks rounds and fires appropriate handler.
