p2p.gossip module¶

p2p.gossip.gossip_message(type, message)¶

Send message to all known nodes over point-to-pont network (broadcast).

Parameters:
  • type (str) – Type of message to process as; unknown types are ignored.
  • message (str) – Payload to deliver to destination to be processed based on type.
p2p.gossip.handle_message(type, message, sender)¶

Used to handle an incoming message sent by another node (heh-heh-heyyyy!).

Parameters:
  • type (str) – Type of message to process as; unknown types are ignored.
  • message (str) – Payload to deliver to subcomponent based on type.
  • sender (str) – Sender of message (primarily used to find key in PKI).
p2p.gossip.send_message(dest, type, message)¶

Send message to destination node over point-to-point network.

Parameters:
  • dest (str) – IP address of receiver.
  • type (str) – Type of message to process as; unknown types are ignored.
  • message (str) – Payload to deliver to destination to be processed based on type.

p2p.synchrony module¶

p2p.synchrony.get_curr_round()¶

Get the current protocol round.

Returns:The integer value of the current round.
Return type:int
p2p.synchrony.is_started()¶

Determine whether a round-based protocol requiring our synchrony assumption has been initiated. Returns a bool representing if the protocol has been started.

p2p.synchrony.log_synchrony()¶

Log protocol execution to stdout.

p2p.synchrony.receive_start_message()¶

Called on receipt of a start message; starts tracking rounds and initializes logging to stdout (see log_synchrony).

p2p.synchrony.round_length = 6¶

The total length of a round, in seconds. This is 3x our synchrony assumption (as per diagram in HW; you may assume the round length is always set to this)

p2p.synchrony.should_send()¶

Determine whether a node should be sending messages when queried. See the PDF on where in the round this falls. Returns True if a node should send, False otherwise.

p2p.synchrony.start_time = None¶

The clock time at which the protocol is started, initialized to None

p2p.synchrony.synchrony_assumption = 2¶

The synchrony assumption (delta in the PDF) to use, in seconds
